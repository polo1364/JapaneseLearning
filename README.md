## 📚 建議學習路徑

### 第一階段：基礎建設（1-2週）

#### 1. 五十音 (Kana)
- 先學**平假名**（あいうえお）
- 再學**片假名**（アイウエオ）
- 每天練習 30 分鐘，用閃卡模式

#### 2. 數字與時間
- 閃卡：🔢 數字與計數
- 閃卡：📅 時間日期
- 這些是對話中最常用到的！

#### 3. 基本問候
- 閃卡：🙌 基本問候
- 練習自我介紹

### 第二階段：生存對話（2-4週）

#### 重點場景練習順序：

| 優先順序 | 場景 | 學習模式 |
|---------|------|----------|
| ⭐⭐⭐ | 便利商店 | 閃卡 → 對話 |
| ⭐⭐⭐ | 餐廳點餐 | 閃卡 → 對話 |
| ⭐⭐⭐ | 車站購票 | 閃卡 → 對話 |

| ⭐ | 藥局買藥 | 閃卡 → 對話 |

#### 每個場景的學習步驟：
1. **閃卡模式**：先熟悉該場景的關鍵單字
2. **聽力模式**：練習聽懂常用句
3. **對話模式**：實際練習對話
   - 先用中文輸入，學習日文說法
   - 再用日文練習，獲得評分反饋

---

### 第三階段：文法強化（持續）

#### 必學文法（按順序）：
1. は・が 的差別
2. です・ます 禮貌形
3. て形（連接、請求）
4. ない形（否定）
5. た形（過去式）
6. 〜たい（想要）
7. 敬語入門

## 💬 對話練習技巧

### 使用 AI 對話的最佳方式：

#### 1. 中文輸入模式（學習階段）
- 用中文說你想表達的
- AI 會教你日文怎麼說
- 注意看「其他說法」和「文法說明」
- 跟著音頻練習發音

#### 2. 日文輸入模式（練習階段）
- 嘗試用日文回答
- 注意評分和具體反饋
- 如果有錯誤，看「正確說法」
- 使用「下一步建議」繼續對話

#### 3. 語音輸入
- 點擊麥克風練習口說
- 觀察音量指示器確認收音
- 說完會自動發送

## 🔑 實用對話模板

### 萬用句（背起來！）

| 情境 | 日文 | 意思 |
|-----|------|------|
| 開場 | すみません | 不好意思 |
| 不懂時 | もう一度お願いします | 請再說一次 |
| 確認 | これでいいですか？ | 這樣可以嗎？ |
| 感謝 | ありがとうございます | 非常感謝 |
| 道歉 | すみません、日本語があまり... | 抱歉，我日文不太... |
| 請求 | ～をお願いします | 請給我～ |
| 結帳 | お会計お願いします | 請結帳 |

### 購物必備

| 日文 | 意思 |
|------|------|
| いくらですか？ | 多少錢？ |
| これください | 我要這個 |
| カードで払えますか？ | 可以刷卡嗎？ |
| 袋はいりません | 不需要袋子 |
| レシートください | 請給我收據 |

### 餐廳必備

| 日文 | 意思 |
|------|------|
| ○○人です | ○○位 |
| メニューをお願いします | 請給我菜單 |
| おすすめは何ですか？ | 推薦什麼？ |
| これは何ですか？ | 這是什麼？ |
| お水ください | 請給我水 |

### 交通必備

| 日文 | 意思 |
|------|------|
| ○○駅はどこですか？ | ○○站在哪裡？ |
| ○○行きの電車は？ | 往○○的電車？ |
| 何番線ですか？ | 幾號月台？ |
| 次の駅は？ | 下一站是？ |

## 🎮 功能使用說明

### 學習模式
- 🃏 **閃卡**：記憶單字，空白鍵翻轉
- 📝 **測驗**：選擇題測驗
- ⌨️ **打字**：練習拼寫
- 🎧 **聽力**：聽音選意
- 📖 **文法**：文法說明和練習
- 💬 **對話**：AI 角色扮演

### 快捷鍵
- `空白鍵`：翻轉閃卡 / 播放發音
- `1-4`：選擇答案
- `S`：慢速播放
- `R`：重複播放
